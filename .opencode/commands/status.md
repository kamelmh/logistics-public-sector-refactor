# status

Quick project health overview: git branch, last commit, uncommitted files, pipeline scripts status, last verify timestamp, and config summary.

## Usage
```
OpenCode status
```

## What it shows
- Git repo status (branch, last commit, uncommitted files)
- Build/verify script availability
- Latest verify result filename
- Config module count
- All key paths

## Implementation
`:status` label in `OpenCode.bat`
