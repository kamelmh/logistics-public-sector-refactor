### ⚙️ The Excel 2010 "Smart Tag" VBA Macro

Press `ALT + F11` in your ERP file, go to **Insert \> Module**, and paste this code. It is designed to read the last "IN" movement and generate the tag.

```
Sub GenererEtiquetteQR_Excel2010()
    Dim wsMouv As Worksheet
    Dim wsTag As Worksheet
    Dim lastRow As Long
    Dim codeArticle As String, refDoc As String, qty As String
    Dim qrData As String, qrURL As String
    Dim qrImage As Picture
    Dim pdfPath As String
    
    ' 1. Set the working sheets
    Set wsMouv = ThisWorkbook.Sheets("MOUVEMENTS")
    Set wsTag = ThisWorkbook.Sheets("ReportTemplate") ' Create this sheet if you haven't yet
    
    ' 2. Find the last entry in the MOUVEMENTS ledger
    lastRow = wsMouv.Cells(wsMouv.Rows.Count, "A").End(xlUp).Row
    
    ' 3. Extract the data (Col B = Code, Col E = Qty, Col H = N° Bon)
    codeArticle = wsMouv.Cells(lastRow, 2).Value
    qty = CStr(wsMouv.Cells(lastRow, 5).Value)
    refDoc = wsMouv.Cells(lastRow, 8).Value
    
    ' 4. Push data to the printable tag template
    wsTag.Range("B2").Value = codeArticle
    wsTag.Range("B6").Value = qty
    wsTag.Range("B7").Value = refDoc
    
    ' 5. Excel 2010 Safe URL Encoding (Replacing spaces to avoid errors)
    qrData = codeArticle & "-Qty:" & qty & "-Bon:" & refDoc
    qrData = Replace(qrData, " ", "%20")
    
    ' 6. Generate the QR Code via free API
    qrURL = "https://api.qrserver.com/v1/create-qr-code/?size=120x120&data=" & qrData
    
    ' 7. Clear old QR codes and insert the new one
    For Each qrImage In wsTag.Pictures
        qrImage.Delete
    Next qrImage
    
    With wsTag.Pictures.Insert(qrURL)
        .Top = wsTag.Range("D4").Top
        .Left = wsTag.Range("D4").Left
    End With
    
    ' 8. Save as PDF in your Dropbox project folder
    pdfPath = ThisWorkbook.Path & "\Etiquette_Reception_" & refDoc & ".pdf"
    
    wsTag.ExportAsFixedFormat _
        Type:=xlTypePDF, _
        Filename:=pdfPath, _
        Quality:=xlQualityStandard, _
        OpenAfterPublish:=True
        
    MsgBox "L'étiquette QR a été générée avec succès pour Excel 2010!", vbInformation
End Sub
```
