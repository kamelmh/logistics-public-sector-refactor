# Claude Thesis Final Audit & Formatting Review Protocol
# Version: v13.2 (Golden State)

## Goal
Perform a final, zero-defect audit of the thesis before official submission. The focus is on linguistic perfection, structural consistency, and absolute alignment with Ground Truth.

## 1. Structural Audit Checklist
- [ ] **Bismillah Page:** Verify it is the absolute first page, centered, 36pt, and has NO page number.
- [ ] **Page Numbering Transition:** 
    - Front matter (TOC, LOT, Abstract) must use Abjad letters (أ، ب، ج...).
    - Main body (Chapter 1 onwards) must use Arabic numerals (1, 2, 3...).
    - Verify the transition happens exactly at the start of "الفصل الأول".
- [ ] **Heading Hierarchy:** Ensure no levels are skipped (H1 -> H2 -> H3).
- [ ] **Table/Figure Captions:** Verify all tables have captions and are listed in the LOT.

## 2. Technical & Data Audit (The "Law" Check)
- [ ] **ART-001 (Toner G030):** Verify all mentions of ART-001 use the correct name and values (D=1546, Q*=176, ROP=212.4, SS=200).
- [ ] **ART-002 (Paper A4):** Verify correct mapping.
- [ ] **ART-005 (Stapler/Agrafeuse):** Verify it is NOT listed as Toner or Paper.
- [ ] **Reference Integrity:** Ensure every inline citation has a matching bibliography entry and every footnote is formatted as "راجع: [Full Cite]".

## 3. Linguistic & Stylistic Polish
- [ ] **Arabic MSA Flow:** Replace generic verbs with academic ones (e.g., use "ترمي إلى" instead of "تهدف إلى", "تجسد" instead of "تظهر").
- [ ] **Consistency:** Ensure terminology (like "نظام دعم القرار") is used consistently throughout.
- [ ] **Tone:** Maintain a formal, third-person academic tone.

## 4. Delivery Format for Edits
When suggesting changes, provide them in the following "Surgical Edit" format:
- **Location:** [Chapter/Section/Line]
- **Old Text:** "[Exact original text]"
- **New Text:** "[Proposed polished text]"
- **Reason:** [Brief academic justification]
