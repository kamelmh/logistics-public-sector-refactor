▸  أولاً — ترقيم الوثائق الإدارية
القاعدة العامة: رمز الوثيقة — السنة — الرقم التسلسلي
DA DA-SSSS-NNN DA-2026-001 قاعدة ثابتة / Format : طلب الاحتياج الداخلي — يصدر عن المصلحة الطالبة قبل أي عملية شراء. SSSS=السنة. NNN=رقم متسلسل يبدأ من 001 كل سنة.
BC BC-SSSS-NNN BC-2026-007 قاعدة ثابتة / Format : أمر الشراء — يصدر عن المديرية للمورد بعد اعتماد DA. يُحال دائماً لرقم DA في الخانة المخصصة.
BR BR-SSSS-NNN BR-2026-003 قاعدة ثابتة / Format : وصل الاستلام — يُحرَّر عند وصول البضاعة من المورد. يُحال لرقم BC المقابل. يُحفَظ في ملف اللوازم.
BS BS-SSSS-NNN BS-2026-012 قاعدة ثابتة / Format : وصل الخروج/الصرف — يُحرَّر عند تسليم المواد للمصلحة الطالبة. يُحال لرقم DA عند الإمكان.
BC_Comp BC-2026-007-C — قاعدة ثابتة / Format : إضافة حرف C (Complément) لأوامر الشراء التكميلية لنفس الطلبية — نادر الاستخدام.
▸  ثانياً — ترميز المواد (Code Article)
ART ART-NNN ART-001 قاعدة ثابتة / Format : كل مادة تأخذ رمزاً فريداً لا يتغير أبداً بعد تسجيلها في النظام. NNN يبدأ من 001 ويزيد بـ 1 لكل مادة جديدة. لا تُعيد استخدام رمز مادة محذوفة.
INC INC-01 INC-01 قاعدة ثابتة / Format : للمواد غير المعرَّفة أو التي لم تُصنَّف بعد. تُعالَج قبل أي عملية إدخال رسمية.

MODÈLE DE WILSON — EOQ  [OPT-03 | D Auto-annualisé | ART-001 | Excel 2010]
D Demande annuelle ART-001  [AUTO annualisé ×250 j] 2,900 unités/an
CL Coût passation par commande (DA)  [modifiable] 50 DA/cmd
PU Prix unitaire ART-001 (DA)  [confirmé terrain] 400 DA/rame
t Taux de possession annuel 0.20 %
SS Stock de sécurité (unités)  [modifiable] 200 unités
LT Délai livraison Oran→Bayadh (jours ouvrés) 2 jours
RÉSULTATS DU MODÈLE  —  Q*, N, ROP, CVT
EOQ Quantité économique Q* = √(2·D·CL / Cs)  où Cs = PU·t 60 unités
N Nombre de commandes par an  N = D / Q* 48 cmd/an
ROP Point de réappro  ROP = SS + CMJ·LT  [CMJ=D/250] 223 unités
Cp Coût de passation annuel  Cp = (D/Q*)·CL 2,417 DA/an
Cd Coût de détention annuel  Cd = (Q*/2+SS)·Cs 18,400 DA/an
CVT Coût Variable Total  CVT = Cp + Cd 20,817 DA/an
📌  EOQ est calculé pour ART-001 (رزمة A4) uniquement, article de classe A.  D est auto-annualisé depuis MOUVEMENTS (38j obs. × 250j ouvrables).  Valeurs attendues : EOQ ≈ 60 rames | ROP ≈ 216 rames | CVT ≈ 4817 DA/an

Sub ProcesserTransaction(TypeMvt As String, CodeArt As String, Qty As Long, RefDoc As String, Notes As String) Dim ws As Worksheet
Set ws = ThisWorkbook.Sheets("MOUVEMENTS")
Dim lr As Long : lr = ws.Cells(ws.Rows.Count,"A").End(xlUp).Row + 1
' --- VALIDATION: block negative stock on OUT ---
If TypeMvt = "OUT" Then
   Dim sIN As Long, sOUT As Long
   sIN  = Application.WorksheetFunction.SumIfs(ws.Range("E3:E500"),ws.Range("B3:B500"),CodeArt,ws.Range("D3:D500"),"IN")
   sOUT = Application.WorksheetFunction.SumIfs(ws.Range("E3:E500"),ws.Range("B3:B500"),CodeArt,ws.Range("D3:D500"),"OUT")
   If Qty > (sIN - sOUT) Then
       MsgBox "Stock insuffisant!" & Chr(13) & "Actuel: " & (sIN-sOUT) & " | Requested: " & Qty, vbCritical, "Sortie refusee"
       Exit Sub
   End If
End If
' --- WRITE static values ---
ws.Cells(lr,1).Value = Date : ws.Cells(lr,2).Value = CodeArt
ws.Cells(lr,4).Value = TypeMvt : ws.Cells(lr,5).Value = Qty
ws.Cells(lr,8).Value = RefDoc  : ws.Cells(lr,13).Value = Notes
' --- COPY expanding SUMPRODUCT formulas from previous row ---
Dim pr As Long : pr = lr - 1
If pr >= 3 Then
   Dim fJ As String, fK As String, fL As String
   fJ = Replace(Replace(ws.Cells(pr,10).Formula, "$B"&pr, "$B"&lr), "$D"&pr, "$D"&lr)
   fJ = Replace(fJ, "$E"&pr, "$E"&lr)
   fK = Replace(Replace(ws.Cells(pr,11).Formula, "$B"&pr, "$B"&lr), "$D"&pr, "$D"&lr)
   fK = Replace(Replace(fK, "$E"&pr, "$E"&lr), "$F"&pr, "$F"&lr)
   fL = Replace(Replace(ws.Cells(pr,12).Formula, "$B"&pr, "$B"&lr), "$D"&pr, "$D"&lr)
   fL = Replace(fL, "$E"&pr, "$E"&lr)
   ws.Cells(lr,3).Formula  = ws.Cells(pr,3).Formula
   ws.Cells(lr,6).Formula  = ws.Cells(pr,6).Formula
   ws.Cells(lr,7).Formula  = "=E" & lr & "*F" & lr
   ws.Cells(lr,9).Formula  = ws.Cells(pr,9).Formula
   ws.Cells(lr,10).Formula = fJ
   ws.Cells(lr,11).Formula = fK
   ws.Cells(lr,12).Formula = fL
End If
Call JournaliserAction("MOUVEMENT_" & TypeMvt, CodeArt & " | Qty:" & Qty & " | Ref:" & RefDoc)
Application.Calculate
MsgBox "Recorded: " & TypeMvt & " | " & CodeArt & " | Qty: " & Qty, vbInformation, "ERP v8.1"
End Sub
